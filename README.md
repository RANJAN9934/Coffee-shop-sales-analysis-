# Coffee-shop-sales-analysis
# ☕ Coffee Shop Sales Analysis (Excel + Power Query)

## 📊 Project Overview
This project focuses on analyzing coffee shop sales data using **Microsoft Excel** and **Power Query**. The goal is to clean raw data, transform it into a structured format, and build an interactive dashboard to generate meaningful business insights.

---

## 🛠 Tools & Technologies
- Microsoft Excel  
- Power Query (Data Cleaning & Transformation)  
- Pivot Tables & Charts  
- Dashboard Design  

---

## 🔧 Data Cleaning (Power Query)
The raw dataset was cleaned and transformed using Power Query:
- Removed duplicates and handled missing values  
- Standardized column names and formats  
- Converted data types (Date, Time, Numeric)  
- Created new columns (Hour, Day, Month)  
- Filtered and structured data for analysis  

---

## 📈 Key Metrics
- **Total Sales:** $698,812  
- **Total Footfall:** 149,116  
- **Average Bill per Person:** $4.69  
- **Average Orders per Person:** 1.44  

---

## 🔍 Key Insights
- Peak sales hours are between **8 AM – 10 AM**  
- Coffee category contributes the highest sales (~39%)  
- Most customers prefer **Regular and Large sizes**  
- **Top Product:** Barista Espresso  
- Higher sales observed on **Monday, Thursday, and Friday**  
- **Best Performing Location:** Hell’s Kitchen  

---

## 📊 Dashboard Features
- Sales trends by hour  
- Category-wise sales distribution  
- Size-wise order distribution  
- Location-based performance  
- Top 5 products analysis
